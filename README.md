Landing Page institucional desenvolvida para a empresa **Império das Peças**, especializada na comercialização de peças automotivas usadas e seminovas.

O projeto foi desenvolvido com foco em boas práticas de desenvolvimento Front-end, utilizando **HTML5 Semântico**, **CSS3 Modularizado** e **JavaScript Puro (Vanilla JavaScript)**, priorizando organização, responsividade e uma experiência agradável ao usuário.

## 🚀 Funcionalidades

- ✅ Layout totalmente responsivo
- ✅ Menu responsivo para dispositivos móveis
- ✅ Navegação com Scroll Suave
- ✅ Animações durante a rolagem da página
- ✅ Galeria de imagens em formato de Slider
- ✅ Auto Play da galeria
- ✅ Pausa automática da galeria ao posicionar o mouse sobre o componente
- ✅ Estrutura HTML semântica
- ✅ Código organizado por componentes

## 💻 Como executar

1. Clone este repositório:

```bash
git clone https://github.com/SEU-USUARIO/landing-page-imperio-das-pecas.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo:

```text
index.html
```

Não é necessário instalar dependências ou utilizar servidor local.
